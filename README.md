# Escape Room VR

A virtual reality escape room experience built in Unity for Oculus. Navigate a fully interactive environment complete with physics-based puzzles, a keypad lock, and immersive UI â€” all playable in VR.

## Tech Stack

- **Engine:** Unity
- **Language:** C#
- **Platform:** Oculus (VR)
- **SDK:** Oculus SDK (OVR)

## Features

- Interact with objects in a fully realized escape room environment
- Solve puzzles including a combination keypad
- VR hand presence and interaction
- Atmospheric audio and UI

## Running the Project

This project requires Unity with the Oculus SDK configured.

1. Clone the repository
2. Open the project in the Unity Editor
3. Ensure the Oculus SDK is properly set up for your headset
4. Open the scene from `Assets/Scenes/`
5. Build and deploy to your Oculus device

> Note: A compatible Oculus headset and the Oculus PC app are required to play.
